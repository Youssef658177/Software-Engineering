##  Learning Path Overview

### 1. Mathematics & Foundations
- **Discrete Math & Logic**  
  - [Mathematics for Computer Science (MIT OCW)](https://ocw.mit.edu/courses/6-042j-mathematics-for-computer-science-fall-2010/) (MIT OCW)  
  - [Linear Algebra – Foundations to Frontiers (edX)](https://www.edx.org/course/linear-algebra-foundations-to-frontiers) (edX)  
  - [Khan Academy – Discrete Mathematics](https://www.khanacademy.org/math/discrete-math) (Khan Academy)

- **Calculus & Probability**  
  - [Calculus One (Coursera)](https://www.coursera.org/learn/calculus1) (Coursera)  
  - [Calculus Two: Sequences and Series (OSSU)](https://ossu.thinkific.com/courses) *(Module from OSSU)* 0  
  - [Introduction to Probability (MIT OCW)](https://ocw.mit.edu/courses/res-6-012-introduction-to-probability-spring-2018/)

---

### 2. Programming & Computing Fundamentals
- **Intro to CS & Programming**  
  - [CS50P: Introduction to Programming with Python (Harvard)](https://cs50.harvard.edu/python/) (edX/Harvard) 1  
  - [MITx – Introduction to CS and Programming Using Python (MIT OCW)](https://ocw.mit.edu/courses/6-100l-introduction-to-cs-and-programming-using-python-fall-2022/) (MIT OCW) 2  
  - [OSSU Intro CS Module (GitHub)](https://github.com/ossu/computer-science) 3

- **Core Programming Concepts**  
  - [How to Code: Simple & Complex Data (OSSU)](https://ossu.thinkific.com/courses) *(OSSU Core CS Programming)* 4  
  - [SICP – Structure and Interpretation of Computer Programs](https://teachyourselfcs.com/) *(via TeachYourselfCS guide)* 5  
  - [freeCodeCamp Curriculum](https://www.freecodecamp.org/) (freeCodeCamp)

---

### 3. Systems & Architecture
- **Computer Architecture & Hardware**  
  - [Build a Modern Computer from First Principles: Nand to Tetris (edX)](https://www.edx.org/course/build-a-modern-computer-from-first-principles-nand2tetris-part-i) (Coursera/edX via OSSU) 6  
  - [Operating Systems: Three Easy Pieces (OSTEP)](http://pages.cs.wisc.edu/~remzi/OSTEP/) (free)  
  - [Computer Systems: A Programmer’s Perspective (book/course recommendation)](https://teachyourselfcs.com/) *(TeachYourselfCS)* 7

- **Networking & Systems Programming**  
  - [Introduction to Computer Networking (OSSU)](https://ossu.thinkific.com/courses) 8  
  - [freeCodeCamp – API & Backend Development](https://www.freecodecamp.org/) (freeCodeCamp)

---

### 4. Algorithms, Databases & Theory
- **Algorithms & Data Structures**  
  - [Algorithms Specialization (Coursera)](https://www.coursera.org/specializations/algorithms) (Coursera)  
  - [The Algorithm Design Manual (Skiena)](https://teachyourselfcs.com/) *(TeachYourselfCS recommendation)* 9

- **Databases**  
  - [Databases Course (OSSU)](https://ossu.thinkific.com/courses) 10  
  - [Readings in Database Systems (Berkeley)](https://teachyourselfcs.com/) *(TeachYourselfCS)* 11

- **Theory & Automata**  
  - [Automata Theory (OSSU)](https://ossu.thinkific.com/courses) 12

---

### 5. Advanced Topics & Applications
- **Machine Learning, Crypto, Graphics, Capstone**  
  - [Machine Learning Course (OSSU)](https://ossu.thinkific.com/courses) 13  
  - [Cryptography I (OSSU)](https://ossu.thinkific.com/courses) 14  
  - [Computer Graphics (OSSU)](https://ossu.thinkific.com/courses) 15

- **Advanced CS Modules**  
  - [Compilers / Software Debugging / Design (OSSU Advanced CS)](https://ossu.thinkific.com/courses) 16  
  - [Distributed Systems & Cloud (OSSU)](https://ossu.thinkific.com/courses) 17
